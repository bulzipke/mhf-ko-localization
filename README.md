# Monster Hunter Frontier Korean Localization
본 프로젝트는 Erupe 서버에서 구동되는 `몬스터 헌터 프론티어(Monster Hunter Frontier)`를 한국어로 즐기기 위한 커뮤니티 기반 한글화 프로젝트입니다.

## 📌 소개
본 패치는 일본어판 텍스트 데이터를 기반으로 하며, 과거 한게임 서비스 당시의 번역 데이터도 참고하여 제작되었습니다. 유저들이 더 편안하게 게임을 즐길 수 있도록 텍스트, UI, 아이템 명칭 등을 한국어로 현지화하는 것을 목표로 합니다.
